# Java Raycasting Engine

A software raycasting engine written entirely in Java without using OpenGL or external game engines.

## Features

- DOOM style
- Textured wall rendering
- Floor and ceiling casting
- Sprite rendering
- Animated enemies
- Weapon animations
- Collision detection
- Particle effects
- Sound effects

## Controls

- Z / S : Move forward/backward
- Q / D : Strafe
- Mouse : Look around
- LMB : Shoot
- 1 / 2 : Switch weapons
